# Ｃ++的位运算

> 作者为蒟蒻高中牲，若有不妥之处请多多包含，并望于评论区指正，不胜感激！

Ｃ++中有**6种位运算符**：

- `&`: 与

  `1 & 1 = 1`

  `1 & 0 = 0`

  `0 & 0 = 0`

- `|`: 或

  `1 | 1 = 1`

  `1 | 0 = 1`

  `0 | 0 = 0`

- `~`: 非

  `~ 1 = 0`

  `~ 0 = 1`

- `^`: 异或

  `1 ^ 1 = 1`

  `1 ^ 0 = 0`

  `0 ^ 0 = 1`

- `<<`: 左移

  $1 << 1 = 10_２$

  $1 << 2 = 100_2$

- `>>`: 右移

  $11_2 >> 1 = 110_2$
